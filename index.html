使用提示 · Tips

Ollama（推荐优先） — 完全本地运行，无需 API Key。安装后运行 ollama pull qwen3.5:35b 即可使用。模型列表自动同步，新安装的模型会实时显示。

DeepSeek — 访问 platform.deepseek.com 获取 API Key。推荐模型：deepseek-chat。

Kimi (Moonshot) — 访问 platform.moonshot.cn 获取 API Key。推荐模型：moonshot-v1-8k。

MiniMax — 访问 platform.minimaxi.com 获取 API Key。推荐模型：MiniMax-Text-01。

GLM (智谱) — 访问 open.bigmodel.cn 获取 API Key。推荐模型：glm-4-flash（免费）。

⚠️ API Key 保存在服务器本地文件中，不会上传到任何外部服务。请不要在公开环境中使用含有敏感 Key 的配置文件。
